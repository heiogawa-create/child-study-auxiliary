# お友だち紹介制度（運用案）

## 利用者向け表示

- 紹介リンクから登録したお友だちの月額決済が完了するたび、紹介者に毎月100円を付与します。
- 報酬は月末締めで集計し、翌月7日までに登録メールアドレス宛へPayPay受取リンクを送ります。
- 報酬は1段階のみです。紹介された方がさらに紹介した分は、最初の紹介者の報酬にはなりません。
- 自己紹介、同一人物による重複登録、不正と判断した登録、返金、チャージバック、未払いの月は対象外です。
- PayPay受取リンクの受取期限切れなどで受領できなかった場合は、運営へ連絡して再発行を依頼できます。

## 管理者の月次手順

1. 毎月1日00:05（日本時間）に前月分を自動で締めます。
2. 会員画面の「月末PayPay精算」で対象月を開きます。
3. 紹介者ごとの登録メール、PayPay ID（任意）、件数、送金額を確認します。
4. PayPayで受取リンクを作り、登録メール宛へ手動送信します。
5. 送金後に「送金済みにする」を押し、日時と金額を保存します。
6. `review` 件数がある場合は、送金後の返金・チャージバックなので個別確認します。

公開前に、利用規約・特定商取引法に基づく表記・プライバシーポリシーと整合するよう、専門家による確認を推奨します。
